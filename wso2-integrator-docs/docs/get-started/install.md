# Install VS Code Extension

Get your development environment ready in under 5 minutes.

## Prerequisites

- **VS Code** 1.80 or later
- **Java** 17 or later (OpenJDK recommended)
- **Operating system**: macOS, Windows, or Linux

## Install the Extension

1. Open VS Code
2. Go to the Extensions panel (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for "WSO2 Integrator"
4. Click **Install**

## Verify Installation

1. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Type "WSO2 Integrator" — you should see available commands
3. Select **WSO2 Integrator: Create New Project** to confirm the extension is working
